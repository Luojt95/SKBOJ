## 快速部署方式

### 方式1: GitHub Codespaces（推荐）
1. 打开 https://github.com/Luojt95/SKBOJ
2. 点击绿色 "Code" 按钮
3. 选择 "Codespaces" 标签
4. 点击 "Create codespace on main"
5. 在打开的终端中运行：
   ```bash
   pnpm install
   pnpm build
   ```
6. 项目就运行起来了！

### 方式2: Vercel 直接导入
1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 点击 "New Project"
4. 选择 "Luojt95/SKBOJ" 仓库
5. 添加上面的环境变量
6. 点击 Deploy
